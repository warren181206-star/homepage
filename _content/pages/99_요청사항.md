# 사용 예시

## 요청사항
- 모든 페이지의 골드 색상을 #b78e64에서 #c9a66b로 변경해주세요
- 헤더와 푸터의 패딩을 10px 늘려주세요
- 모바일에서 폰트 크기를 전체적으로 2px 키워주세요

## 대상 페이지
- 전체 (또는 특정 페이지 목록)

# 완료된 요청사항

## 2026-01-20: 헤더 배경색 흰색으로 변경
- 다크(#1a1a1a) → 흰색(#ffffff) 배경으로 변경
- 메뉴 텍스트 색상 #333으로 변경
- 호버 시 골드색(#b78e64)으로 변경
- 하단 라인 골드색으로 표시

## 2026-01-19: 헤더 배경색 다크로 변경
- 로즈 골드 그라데이션 → 다크(#1a1a1a) 배경으로 변경
- 화이트 로고가 잘 보이도록 수정

## 2026-01-12: 블랙 배경 이미지 적용
![[블랙배경.jpg]]
- 모든 페이지에 블랙배경.jpg를 기본 배경으로 적용
- images/common/ 폴더에 저장
- 30% 투명도의 어두운 오버레이 추가
- 모바일에서도 position: fixed로 배경 잘 보이도록 적용 완료
